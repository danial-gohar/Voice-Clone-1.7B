## How to use

1. Click the "Open in Colab" badge next to the notebook you want to use.
2. This will open the notebook in Google Colab.
3. Make sure to connect to a runtime (GPU is usually recommended for AI tasks) by going to **Runtime ‚Üí Change runtime type**.
4. Run the cells in order.
